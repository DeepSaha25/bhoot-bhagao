# Bhoot Bhagao

A peaceful, frontend-only React + Vite website for spiritual reassurance, prayer, calm breathing, and multi-faith unity.

## Tech

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React icons
- Frontend only, no backend

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project Structure

```text
src/
  assets/
  components/
  data/
  hooks/
  pages/
  utils/
```

## Media Notes

The app uses remote, lazy-loaded placeholder images and generated browser audio for ambient previews. Replace entries in `src/data/gallery.js`, `src/data/religions.js`, and `src/data/prayers.json` with licensed images and devotional audio files when publishing.

The site is intentionally peaceful, respectful, and non-horror themed.
